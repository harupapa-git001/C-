これはC#のプログラミングサンプル集です。

拡張子.csと.exeはセットで保存されています。（.csをmcsコマンドでコンパイルして.exeファイルを生成します。)

$ mcs ファイル名

実行ファイル.exeを実行する際にはmonoコマンドを使います。

$ mono 実行ファイル名


Console.ReadLine()はコンソールからのキーボード入力を意味します。
（プロンプトやターミナルからの実行では正常に機能しますが、ブラウザのコンパイルではエラーになります。)

Sample103_1は明示的にエラーを作っていますので.exeファイルは存在しません。

ExampleProblem307は例外を投げられます。
